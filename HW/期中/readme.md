## 核心主題：中央極限定理 (Central Limit Theorem, CLT)

在機率統計中，**中央極限定理**被譽為最重要的定理之一。它解釋了為什麼「正態分佈」（Normal Distribution）在自然界中無處不在。

### 1. 理論白話文

不管你的原始資料長什麼樣子（可能是平的、偏向一邊的、或是奇形怪狀的），只要你每次隨機抽取一組樣本並計算**平均值**，當抽取的次數夠多時，這些**平均值的分布**一定會長得像一個「鐘形」（即正態分佈）。

---

## 2. 程式實作解析 (Python)

這段程式碼分為三個主要模組：

### A. 產生原始母體 (Population)

我們刻意不使用正態分佈，而是使用 **均勻分佈 (Uniform Distribution)**。在均勻分佈中，每個數字出現的機率都一樣，圖形看起來像一個長方形。

```python
data = np.random.uniform(0, 1, 100000) # 產生10萬個0到1之間的隨機數

```

### B. 抽樣實驗 (Sampling)

這是程式的核心。我們進行  次實驗，模擬重複觀測的過程：

* 每次從 10 萬個數據中隨機抽  個。
* 計算這  個人的平均體重（或任何數值）。
* 把這個平均值記錄下來。

```python
for _ in range(num_samples):
    sample = np.random.choice(data, size=sample_size) # 隨機抽樣
    sample_means.append(np.mean(sample))           # 計算並儲存平均值

```

### C. 可視化觀察

使用直方圖（Histogram）來對比「原始數據」與「平均值的數據」。

## 3. 數學公式補充

當樣本數  夠大時，樣本平均數  的分布會滿足：

1. **期望值**：（樣本平均值的平均會等於母體平均）。
2. **標準差（標準誤）**：。這意味著樣本數  越大，波動越小，圖形越窄。

---

## 4. 你會看到的結果與意義

當你運行程式後，你會看到兩張圖：

1. **左圖 (Original Distribution)**：是一個平坦的長方形。這代表原始數據中，0.1、0.5、0.9 出現的機會都一樣。
2. **右圖 (Sample Means Distribution)**：會變成一個漂亮的**對稱鐘形曲線**。

**這告訴我們兩件事：**

* **預測能力**：即使我們不知道個體的行為（隨機且混亂），我們也能精確預測「一群人的平均行為」。
* **誤差控制**：這也是為什麼民調或科學實驗只需要抽樣幾千人，就能推論全體人口的原因。

### 2. 核心統計概念表

| 概念 | 說明 | 程式中的體現 |

### 3. 這個程式在現實中有什麼用？

* **品質控管**：工廠不需要檢查每一顆螺絲，只要每天抽 50 顆取平均值，如果平均值偏離中心太遠，就知道機器出問題了。
* **醫學實驗**：測試新藥時，我們無法讓全世界的人都試藥，但透過對 100 位志願者的「平均效果」進行統計，我們可以推論對全人類的效果。
