# ⭐ 一、夏農信道編碼定理（Shannon Channel Coding Theorem）

又稱：**夏農第二定理（Shannon’s Second Theorem）**
用途：描述 *有雜訊的通道* 仍能可靠傳輸訊息的條件。

---

## 🔑 **核心內容（最重要的兩句話）**

1. **對於任何有雜訊的通道，都存在一些編碼方法，可以讓錯誤率任意小。**
2. **但前提是你的傳輸速率 R 小於通道容量 C：**

[
R < C
]

只要傳輸速率比通道能承受的容量低，就可以用某些（可能非常複雜的）編碼方式讓錯誤趨近 0。

---

## 🧠 容量 (Channel Capacity) 是什麼？

對一個通道：

* 有雜訊
* 雜訊可以破壞訊號
* 但可以透過編碼對抗雜訊

通道「最大能可靠傳輸的資訊量」叫做：

[
C = \text{channel capacity}
]

---

## 📌 敘述更完整的夏農信道編碼定理

* 若傳輸速率 **R < C**：
  → 存在某些編碼法，使錯誤率可以做到 **任意小**（趨近於 0）。

* 若傳輸速率 **R > C**：
  → 不論用什麼編碼，錯誤率一定 **不能降到 0**。

**這個定理告訴我們：有雜訊的世界仍然可以「幾乎無誤」通信。**

---

## 📦 舉例

假設某通道容量是 **C = 1 Mbps**：

* 你以 **0.5 Mbps** 傳資料 → 可以做到錯誤率非常非常小
* 你以 **1.5 Mbps** 傳 → 不可能避免很多錯誤

# ⭐ 二、夏農–哈特利定理（Shannon–Hartley Theorem）

用途：「計算通道容量 C」，也就是：**一條有噪音的實體通訊通道最多能傳多少資訊。**

這個定理是專門針對 **有白雜訊（AWGN）的類比訊號：「寬頻通道」的容量公式**。

---

## 🔑 核心公式

[
C = B \log_2 \left( 1 + \frac{S}{N} \right)
]

其中：

| 符號      | 意義                                |
| ------- | --------------------------------- |
| ( C )   | 通道容量（每秒可傳多少 bit）                  |
| ( B )   | 通道頻寬（Hz）                          |
| ( S/N ) | 訊號雜訊比（Signal-to-Noise Ratio, SNR） |

---

## 📌 意義

### 1. 通道越寬（B 越大）→ 能傳越多

例如：Wi-Fi 20 MHz → 速度快於 5 MHz。

### 2. 訊號越清楚（S/N 越大）→ 能傳更多

如果雜訊大到淹沒訊號（S/N 很低），容量會下降。

---

## 📦 舉例計算

假設通道：

* 頻寬 ( B = 1 \text{ MHz} )
* 訊號雜訊比 ( S/N = 100 = 20 \text{ dB} )

容量：

[
C = 10^6 \log_2 (1 + 100)
\approx 10^6 \log_2(101)
\approx 6.66 \text{ Mbps}
]

即使你硬要傳 100 Mbps，也是做不到的。

---

# ⭐ 三、兩個定理的關係（很重要）

| 定理           | 功能                          | 類型  |
| ------------ | --------------------------- | --- |
| **夏農信道編碼定理** | 告訴你：若速率 < 通道容量，就可做到錯誤率趨近 0。 | 理論性 |
| **夏農–哈特利定理** | 告訴你：實體通道容量 C 要怎麼算。          | 實作性 |

順序為：

1. **先用夏農–哈特利定理算出通道容量 C**
2. **再用夏農信道編碼定理知道最大能用的傳輸速率 R < C**

---

# ⭐ 四、總結（最精簡）

### 夏農信道編碼定理

> 「傳輸速率低於通道容量，就能做到幾乎無誤。」
> （給你「能力界線」）

### 夏農–哈特利定理

> 「如何計算通道容量 C。」
> （給你「界線的數值」）

兩者結合，就是現代所有：

* Wi-Fi
* 4G / 5G
* 衛星通信
* 光纖
* 壓縮與編碼

…的理論基礎。
